Phase / status time mix

i

Это секция со средним временем нахождения задач в каждом статусе внутри Cycle time.

В неё попадают только статусы, которые лежат в рабочем участке workflow от стартовых статусов цикла до Done, при этом сами статусы из Done statuses исключаются.

Дни считаются по истории переходов задач: для каждой задачи берётся точное время между переходами по статусам внутри окна Cycle time, затем по каждому статусу считается среднее значение по задачам из выбранного диапазона дат.

Цвета рассчитываются относительно текущего списка статусов: близкие к типичным значения подсвечиваются зелёным, более высокие уходят в жёлтый, а самые выбивающиеся большие значения становятся красными.
